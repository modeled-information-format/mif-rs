---
title: "Topic binds an uncataloged or version-mismatched ontology"
description: "A topic directly binds an ontology id that is not cataloged, or pins a version that does not match the cataloged one."
---

**Type URI:** `https://modeled-information-format.github.io/mif-rs/references/errors/direct-binding-invalid/v1`
**HTTP status:** 422
**Exit code:** 1
**Emitted by:** `mif-rh` (`MifRhError::DirectBindingInvalid`)

## Message

```
topic '{topic}' binds ontology '{id}', which is not cataloged or version-mismatched
```

## Cause

This fires when a topic directly binds an ontology id that either does not
appear in the catalog at all, or pins a version that does not match the
version the catalog actually has cataloged for it. Either way, the binding
cannot be resolved against what's on record.

## Suggested fix

**MaybeIncorrect** — catalog the missing ontology, correct its pinned version, or remove the invalid topic binding, then retry.
