---
title: "Report failed schema validation"
description: "A projected report failed validation against its schema."
---

**Type URI:** `https://modeled-information-format.github.io/mif-rs/references/errors/schema-validation-failed/v1`
**HTTP status:** 422
**Exit code:** 1
**Emitted by:** `mif-rh` (`MifRhError::SchemaValidationFailed`)

## Message

```
{path} failed schema validation against {schema_path}: {detail}
```

## Cause

This fires when a projected report failed validation against its schema.
The report file was read and projected successfully, but its resulting
shape does not conform to what the schema requires.

## Suggested fix

**MaybeIncorrect** — fix the report to conform to its schema (see the listed validation errors), then retry.
