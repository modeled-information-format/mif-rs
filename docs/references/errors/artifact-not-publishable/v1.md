---
title: "Synthesized artifact has no publishable content"
description: "A synthesized artifact has no surviving findings or citations, so there is nothing publishable to render."
---

**Type URI:** `https://modeled-information-format.github.io/mif-rs/references/errors/artifact-not-publishable/v1`
**HTTP status:** 422
**Exit code:** 1
**Emitted by:** `mif-rh` (`MifRhError::ArtifactNotPublishable`)

## Message

```
artifact from {path} has no publishable content (no surviving findings, or no citations to cite)
```

## Cause

This fires when a synthesized artifact ends up with no sections, no finding
references, and no sources — there is simply nothing publishable to render.
It typically means the findings directory either has no surviving findings
or none of the surviving findings carry a citation to cite.

## Suggested fix

**MaybeIncorrect** — ensure at least one surviving finding carries a citation before synthesizing.
