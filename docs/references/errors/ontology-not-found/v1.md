---
title: "Ontology not found in the supplied corpus"
description: "The requested ontology ID was not found in the supplied corpus."
---

**Type URI:** `https://modeled-information-format.github.io/mif-rs/references/errors/ontology-not-found/v1`
**HTTP status:** 404
**Exit code:** 3
**Emitted by:** `mif-ontology` (`OntologyError::NotFound`)

## Message

```
ontology '{0}' not found in the supplied corpus
```

## Cause

`resolve_chain` was asked to resolve an ontology ID that does not exist in
the supplied corpus — either the requested ID itself, or one of its
transitive `extends` parents.

## Suggested fix

**MaybeIncorrect** — add the missing ontology definition to the supplied
corpus directory, or correct the requested ontology ID, then retry.
