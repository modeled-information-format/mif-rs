title: "Internal schema compilation error"
description: "mif-schema's vendored MIF schema failed to compile."
---

**Type URI:** `https://modeled-information-format.github.io/mif-rs/references/errors/schema-compilation/v1`
**HTTP status:** 500
**Exit code:** 1
**Emitted by:** `mif-schema` (`MifSchemaError::SchemaCompilation`)

## Message

```
internal error: vendored MIF schema failed to compile: {0}
```

## Cause

The vendored schema itself (`mif.schema.json`, `citation.schema.json`,
`ontology.schema.json`, or `definitions/entity-reference.schema.json`) failed
to compile via `jsonschema`. This indicates a bug in this crate's vendored
schema files, not a problem with the document, citation, or ontology
definition being validated.

## Suggested fix

**Unspecified** — this indicates a bug in `mif-schema`'s vendored schema
files. Report it upstream.
