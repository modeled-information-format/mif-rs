---
title: "Ontology definition failed schema validation"
description: "An ontology definition file did not conform to ontology.schema.json."
---

**Type URI:** `https://modeled-information-format.github.io/mif-rs/references/errors/invalid-ontology-definition/v1`
**HTTP status:** 422
**Exit code:** 2
**Emitted by:** `mif-ontology` (`OntologyError::Invalid`)

## Message

```
{path} failed ontology schema validation: {source}
```

## Cause

The ontology definition file at `path` parsed as valid YAML, but failed
schema validation against the canonical `ontology.schema.json` (wrapping a
`mif_schema::MifSchemaError`).

## Suggested fix

**MaybeIncorrect** — correct the ontology definition so it conforms to
`ontology.schema.json`, then retry.
