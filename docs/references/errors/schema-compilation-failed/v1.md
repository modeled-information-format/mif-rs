---
title: "Report schema failed to compile"
description: "A report schema, or one of its $ref dependencies, could not be compiled into a validator."
---

**Type URI:** `https://modeled-information-format.github.io/mif-rs/references/errors/schema-compilation-failed/v1`
**HTTP status:** 422
**Exit code:** 2
**Emitted by:** `mif-rh` (`MifRhError::SchemaCompilation`)

## Message

```
failed to compile schema {path}: {detail}
```

## Cause

This fires when a report schema, or one of its `$ref` dependencies, was not
valid JSON, or when the `jsonschema` validator could not be compiled from
it. It surfaces schema-authoring problems before any document is ever
validated against the schema.

## Suggested fix

**MaybeIncorrect** — fix the schema (or its $ref dependency) so it is valid JSON Schema, then retry.
