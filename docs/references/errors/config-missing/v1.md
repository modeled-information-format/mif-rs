---
title: "Harness config file does not exist"
description: "The harness.config.json file could not be found at the given path."
---

**Type URI:** `https://modeled-information-format.github.io/mif-rs/references/errors/config-missing/v1`
**HTTP status:** 404
**Exit code:** 2
**Emitted by:** `mif-rh` (`MifRhError::ConfigMissing`)

## Message

```
config file {path} does not exist
```

## Cause

This fires when the config file (`harness.config.json`) is missing at the
path the caller supplied. It is a straightforward existence check, not a
parse or validation failure — the file simply is not there.

## Suggested fix

**MaybeIncorrect** — supply the correct `--config` path to `harness.config.json`, then retry.
