---
title: "No document with the given id has been ingested"
description: "A mif-cli find-similar query named an id that has never been ingested."
---

**Type URI:** `https://modeled-information-format.github.io/mif-rs/references/errors/mif-cli-document-not-found/v1`
**HTTP status:** 404
**Exit code:** 3
**Emitted by:** `mif-cli` (`CliError::DocumentNotFound`)

## Message

```
no document with id '{0}' has been ingested into this vector store
```

## Cause

`mif-cli find-similar` was asked for documents similar to an anchor `id`
that has never been ingested into the target vector store.

## Suggested fix

**MaybeIncorrect** — ingest the document first with `mif-cli ingest`, or
check the id for a typo.
