## Message

```
failed to initialize the Hugging Face Hub client: {0}
```

## Cause

`hf_hub::api::sync::ApiBuilder::build()` failed while initializing the
Hugging Face Hub API client used to fetch the embedding model.

## Suggested fix

**Unspecified** — this indicates an internal problem with `mif-embed` or the
cached model files. Try clearing the model cache directory and retrying, or
report it upstream if the problem persists.
