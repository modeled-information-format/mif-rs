---
title: "Another review is already in progress"
description: "Another mif-rh-cli review run already holds the exclusive review lock."
---

**Type URI:** `https://modeled-information-format.github.io/mif-rs/references/errors/lock-held/v1`
**HTTP status:** 409
**Exit code:** 2
**Emitted by:** `mif-rh` (`MifRhError::LockHeld`)

## Message

```
another review is already in progress (lock held by pid {holder_pid})
```

## Cause

Another `review` run already holds the lock.

## Suggested fix

**MaybeIncorrect** — wait for the in-progress review to finish, then retry.
