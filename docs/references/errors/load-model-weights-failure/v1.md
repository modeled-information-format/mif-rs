**Type URI:** `https://modeled-information-format.github.io/mif-rs/references/errors/load-model-weights-failure/v1`
**HTTP status:** 500
**Exit code:** 1
**Emitted by:** `mif-embed` (`EmbedError::Model`)

## Message

```
failed to load model weights: {0}
```

## Cause

Loading the fetched `model.safetensors` weights into the CPU-only `candle`
BERT model architecture failed.

## Suggested fix

**Unspecified** — this indicates an internal problem with `mif-embed` or the
cached model files. Try clearing the model cache directory and retrying, or
report it upstream if the problem persists.
