---
title: "Every finding is falsified — nothing to synthesize"
description: "Every finding in a directory is falsified, leaving nothing to synthesize."
---

**Type URI:** `https://modeled-information-format.github.io/mif-rs/references/errors/no-surviving-findings/v1`
**HTTP status:** 422
**Exit code:** 1
**Emitted by:** `mif-rh` (`MifRhError::NoSurvivingFindings`)

## Message

```
no surviving findings to synthesize in {path}
```

## Cause

This fires during synthesis when every finding in the target directory has
been falsified. Synthesis draws only on findings that survived falsification,
so a directory where none did leaves nothing to build a synthesized artifact
from.

## Suggested fix

**MaybeIncorrect** — nothing to do until at least one finding survives falsification.
