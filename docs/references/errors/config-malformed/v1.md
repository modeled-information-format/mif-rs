title: "Harness config's .ontologies field is not an array"
description: "harness.config.json's .ontologies field exists but is not an array."
---

**Type URI:** `https://modeled-information-format.github.io/mif-rs/references/errors/config-malformed/v1`
**HTTP status:** 422
**Exit code:** 1
**Emitted by:** `mif-rh` (`MifRhError::ConfigMalformed`)

## Message

```
{path}'s .ontologies is not an array: {detail}
```

## Cause

`harness.config.json`'s `.ontologies` field exists but is not a JSON array.
Rather than guessing how to coerce or append to a field of the wrong shape,
a discovered ontology cannot be safely written into it.

## Suggested fix

**MaybeIncorrect** — fix harness.config.json so its .ontologies field is an array, then retry.
