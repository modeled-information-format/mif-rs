---
title: "Failed to open the database or initialize its schema"
description: "Failed to open the SQLite database or initialize its schema."
---

**Type URI:** `https://modeled-information-format.github.io/mif-rs/references/errors/open-database-failure/v1`
**HTTP status:** 500
**Exit code:** 1
**Emitted by:** `mif-store` (`StoreError::Open`)

## Message

```
failed to open database at {path}: {source}
```

## Cause

`SQLite` failed to open the database file at `path`, or to initialize the
`embeddings` table schema on it — e.g. the path is an existing directory, or
points to a non-SQLite file.

## Suggested fix

**Unspecified** — this indicates a corrupt or inaccessible database file.
Verify the file's permissions and integrity, or delete it to let
`mif-store` recreate it.
