---
title: "Expected field was not a string"
description: "A field expected to be a string (e.g. id / @id) was not one."
---

**Type URI:** `https://modeled-information-format.github.io/mif-rs/references/errors/field-not-a-string/v1`
**HTTP status:** 422
**Exit code:** 2
**Emitted by:** `mif-frontmatter` (`FrontmatterError::FieldNotAString`)

## Message

```
field '{field}' is not a string
```

## Cause

A field this crate requires to be a string (typically `id` in
`md_to_jsonld`, or `@id` in `jsonld_to_md`) held a non-string YAML/JSON
value instead (e.g. a number).

## Suggested fix

**MaybeIncorrect** — correct the offending field's type in the document.
