description: "A registry-discovered ontology id is not a bare, lowercase slug."
---

**Type URI:** `https://modeled-information-format.github.io/mif-rs/references/errors/malformed-ontology-id/v1`
**HTTP status:** 422
**Exit code:** 1
**Emitted by:** `mif-rh` (`MifRhError::MalformedOntologyId`)

## Message

```
registry index declares a malformed ontology id: '{id}' (refusing, fail-closed)
```

## Cause

A registry-discovered ontology id is not a bare, lowercase slug. Such an id
is treated as a poisoned or malformed index entry, since a value that isn't
constrained to a simple slug could otherwise escape its intended vendored
directory once written into `harness.config.json`.

## Suggested fix

**Unspecified** — the registry index entry is malformed or unsafe; fix it upstream in the canonical registry before retrying.
