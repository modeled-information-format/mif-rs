---
title: "New version equals the current version"
description: "The requested new version equals the current one."
---

**Type URI:** `https://modeled-information-format.github.io/mif-rs/references/errors/version-unchanged/v1`
**HTTP status:** 422
**Exit code:** 2
**Emitted by:** `mif-rh` (`MifRhError::VersionUnchanged`)

## Message

```
new version equals current ({value}); nothing to bump
```

## Cause

This fires during a version-bump operation when the requested new version
resolves to exactly the same value as the current one. Since a bump is
supposed to move a version forward, there is nothing left for `mif-rh` to
write.

## Suggested fix

**MaybeIncorrect** — pass a different version or bump keyword — the
requested version matches the current one, so there is nothing to bump.
