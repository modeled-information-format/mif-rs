---
title: "Malformed ontology definition YAML"
description: "An ontology definition file was not valid YAML."
---

**Type URI:** `https://modeled-information-format.github.io/mif-rs/references/errors/invalid-yaml/v1`
**HTTP status:** 422
**Exit code:** 2
**Emitted by:** `mif-ontology` (`OntologyError::Yaml`)

## Message

```
failed to parse {path} as YAML: {source}
```

## Cause

The ontology definition file at `path` could not be parsed as YAML at all
(a syntax error), before schema validation is even attempted.

## Suggested fix

**MaybeIncorrect** — fix the YAML syntax error in the ontology definition
file, then retry.
