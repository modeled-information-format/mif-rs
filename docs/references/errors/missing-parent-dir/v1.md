---
title: "Database path's parent directory does not exist"
description: "The parent directory of the requested vector store database path does not exist."
---

**Type URI:** `https://modeled-information-format.github.io/mif-rs/references/errors/missing-parent-dir/v1`
**HTTP status:** 400
**Exit code:** 2
**Emitted by:** `mif-store` (`StoreError::MissingParentDir`)

## Message

```
parent directory {path} of the database path does not exist
```

## Cause

`VectorStore::open` was given a database path whose parent directory does
not exist. `mif-store` never creates it — that is the caller's
responsibility (the database *file* itself is created by SQLite if absent,
but a missing parent directory is reported rather than silently created).

## Suggested fix

**MachineApplicable** — create the parent directory of the database path,
then retry. This is the only problem type in the whole workspace marked
safe for an agent to auto-fix without escalating to a human.
