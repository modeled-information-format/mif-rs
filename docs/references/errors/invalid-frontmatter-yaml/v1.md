**HTTP status:** 422
**Exit code:** 2
**Emitted by:** `mif-frontmatter` (`FrontmatterError::Yaml`)

## Message

```
failed to parse frontmatter as YAML: {source}
```

## Cause

A `---\n...\n---` block was found, but its contents were not valid YAML (a
syntax error).

## Suggested fix

**MaybeIncorrect** — fix the YAML syntax error in the frontmatter.
