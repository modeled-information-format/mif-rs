---
title: "A SQLite query failed"
description: "A SQLite query against the vector store failed."
---

**Type URI:** `https://modeled-information-format.github.io/mif-rs/references/errors/sqlite-query-failure/v1`
**HTTP status:** 500
**Exit code:** 1
**Emitted by:** `mif-store` (`StoreError::Query`)

## Message

```
sqlite query failed: {source}
```

## Cause

A `SQLite` statement (insert, select, or aggregate) failed after the
database was already successfully opened — covers `upsert`, `get`, `count`,
`stats`, and `top_k_similar`.

## Suggested fix

**Unspecified** — this indicates a corrupt or inaccessible database file.
Verify the file's permissions and integrity, or delete it to let
`mif-store` recreate it.
