---
title: "JSON-LD field could not be converted to YAML"
description: "Failed to convert a JSON-LD value back into its YAML representation."
---

**Type URI:** `https://modeled-information-format.github.io/mif-rs/references/errors/field-yaml-conversion-failure/v1`
**HTTP status:** 422
**Exit code:** 2
**Emitted by:** `mif-frontmatter` (`FrontmatterError::YamlConversion`)

## Message

```
failed to convert field '{field}' to YAML: {source}
```

## Cause

`jsonld_to_md` could not convert a JSON-LD field's value back into YAML for
frontmatter serialization.

## Suggested fix

**MaybeIncorrect** — correct the offending field's type in the document.
