---
title: "Frontmatter field could not be converted to JSON"
description: "Failed to convert a YAML value into its JSON-LD representation."
---

**Type URI:** `https://modeled-information-format.github.io/mif-rs/references/errors/field-json-conversion-failure/v1`
**HTTP status:** 422
**Exit code:** 2
**Emitted by:** `mif-frontmatter` (`FrontmatterError::JsonConversion`)

## Message

```
failed to convert field '{field}' to JSON: {source}
```

## Cause

`md_to_jsonld` could not convert a YAML frontmatter field's value into JSON.
Real, spec-legal YAML can express structures JSON's data model cannot
represent — for example, a mapping keyed by a complex key (`? [1, 2] : value`).

## Suggested fix

**MaybeIncorrect** — correct the offending field's type in the document.
