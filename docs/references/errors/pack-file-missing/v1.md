---
title: "Pack is missing a required file or section"
description: "A pack is missing a file bump_version needs (its plugin.json, SKILL.md, or family doc section/row)."
---

**Type URI:** `https://modeled-information-format.github.io/mif-rs/references/errors/pack-file-missing/v1`
**HTTP status:** 422
**Exit code:** 2
**Emitted by:** `mif-rh` (`MifRhError::PackFileMissing`)

## Message

```
pack '{name}': missing or malformed {path}
```

## Cause

This fires when the `bump_version` operation looks for a file it needs from
a pack — its `plugin.json`, its `SKILL.md`, or the relevant section/row in
the pack family's own doc — and finds it either missing or malformed.

## Suggested fix

**MaybeIncorrect** — add the missing file or section the pack's own
conventions require (plugin.json .version, SKILL.md version: frontmatter, or
the family doc's **Version:** row), then retry.
