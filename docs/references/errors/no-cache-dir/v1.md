---
title: "No user cache directory available"
description: "The platform has no resolvable user cache directory to cache the embedding model."
---

**Type URI:** `https://modeled-information-format.github.io/mif-rs/references/errors/no-cache-dir/v1`
**HTTP status:** 500
**Exit code:** 1
**Emitted by:** `mif-embed` (`EmbedError::NoCacheDir`)

## Message

```
no user cache directory available to cache '{model}'
```

## Cause

`dirs::cache_dir()` returned `None` — the platform has no resolvable user
cache directory, so `sentence-transformers/all-MiniLM-L6-v2` has nowhere to
be cached on first use.

## Suggested fix

**Unspecified** — this indicates an internal problem with `mif-embed` or the
cached model files. Try clearing the model cache directory and retrying, or
report it upstream if the problem persists.
