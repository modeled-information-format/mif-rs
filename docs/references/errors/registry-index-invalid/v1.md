---
title: "Ontology registry index is malformed"
description: "The registry index (index.json) was not valid JSON or did not match the expected shape."
---

**Type URI:** `https://modeled-information-format.github.io/mif-rs/references/errors/registry-index-invalid/v1`
**HTTP status:** 502
**Exit code:** 1
**Emitted by:** `mif-rh` (`MifRhError::RegistryIndexInvalid`)

## Message

```
ontology registry index at {registry_source} is malformed: {detail}
```

## Cause

The registry index (`index.json`) was not valid JSON or did not match the
expected shape.

## Suggested fix

**MaybeIncorrect** — the registry source's index.json is not valid; fix it
upstream, or point `MIF_ONTOLOGY_SOURCE` at a known-good mirror, then retry.
